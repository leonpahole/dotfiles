h2. Inštalacija

Datoteki @Slovensko (DE&AT).icns@ in @Slovensko (DE&AT).keylayout@ prenesite v @/Library/Keyboard Layouts/@. Nato se odpravite v System Preferences -> Language & Region -> Keyboard Preferences, zavihek Input Sources, pritisnite +, v seznamu Others najdite *Slovensko (DE/AT)* in pritisnite Add.

Za aktiviranje je potrebno v zgornji vrstici (kjer je ura in datum) izbrati iz seznama *Slovensko (DE/AT)*.

!http://i.imgur.com/AjDxiqC.png!

h2. Urejanje

Če vam razporeditev ni všeč jo lahko sami uredite s pomočjo aplikacije "Ukelele":http://scripts.sil.org/cms/scripts/page.php?site_id=nrsi&id=ukelele.
